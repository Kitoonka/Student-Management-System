# Student Management System

A Java-based Student Management System built with **Apache NetBeans** and **MySQL**.  
This project allows managing student records including registration, updates, and viewing student data.

## Features
- Add, edit, and delete student records
- Search students by ID or name
- Simple and user-friendly GUI using Java Swing
- Data stored in MySQL database
